In the __header__ of each WikiPage of the MindWiki you can define machine readable properties by using the pattern:

{{{
* PropName: value
}}}

MindProps extend the MindWiki towards a SemanticWiki.
MindProps can be evaluated via MindApi by other WebApp's.
